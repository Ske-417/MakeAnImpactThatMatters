# デロイト体験 人生ゲーム / Consulting Experience Life Game

コンサルティング業務を体験できる対戦型ボードゲームです。

## 🎮 ゲームの特徴

### コンサルティング要素
- **プロジェクトマス** 📊: 大型M&A、DX推進、業務改善などのプロジェクトに参加
- **スキルアップマス** 📚: リーダーシップ研修、データ分析講座、戦略立案ワークショップでスキル向上
- **クライアント対応マス** 🤝: 重要顧客との会議、緊急対応、提案コンペで経験を積む
- **休暇マス** ☕: リフレッシュ休暇やワーケーションで英気を養う
- **チャレンジマス** ⚡: 厳しい納期、困難な交渉、チーム危機管理で大きく成長

### プレイヤーの成長
各プレイヤーは3つのスキルを持っています：
- **戦略スキル**: 経営戦略、プロジェクト計画の能力
- **コミュニケーションスキル**: クライアント対応、チームワークの能力
- **分析スキル**: データ分析、問題解決の能力

経験値とスキルを積み重ね、最高のコンサルタントを目指しましょう！

## 🎯 遊び方

1. **プレイヤー設定**: 2〜6人のプレイヤー数を選択し、名前を入力
2. **ゲーム開始**: 「ゲーム開始」ボタンをクリック
3. **サイコロを振る**: 自分のターンに「サイコロを振る」ボタンをクリック
4. **イベント発生**: 特別なマスに止まるとイベントが発生し、経験値やスキルを獲得
5. **ゴールを目指す**: 24マス目のゴールに到達してゲーム終了
6. **勝利判定**: 最も経験値の高いプレイヤーが優勝！

## 📊 マス目の種類

| アイコン | タイプ | 効果 |
|---------|--------|------|
| 📊 | プロジェクト | 経験値+10〜20、戦略・分析スキル向上 |
| 📚 | スキルアップ | 経験値+5、各種スキル+3 |
| 🤝 | クライアント対応 | 経験値+8〜20、コミュニケーション・戦略スキル向上 |
| ☕ | 休暇 | 経験値+3〜5、全スキル向上またはボーナス歩数 |
| ⚡ | チャレンジ | 経験値+15〜20、大幅なスキル向上 |

## 🎨 イベント例

### プロジェクトイベント
- 🏢 大型M&Aプロジェクト: 企業買収のデューデリジェンス
- 💼 デジタル変革プロジェクト: DX推進支援
- 📈 業務改善プロジェクト: プロセス最適化

### スキルアップイベント
- 📚 リーダーシップ研修: チームマネジメント強化
- 🎓 データ分析講座: データサイエンス習得
- 💡 戦略立案ワークショップ: 経営戦略の学習

### クライアント対応イベント
- 🤝 重要顧客との会議: CEOプレゼンテーション成功
- 📞 緊急クライアント対応: トラブル解決
- 🎯 提案コンペ勝利: 競合に勝利して新規受注

### 休暇イベント
- ☕ リフレッシュ休暇: 次ターン+2歩のボーナス
- 🌴 ワーケーション: 新しいアイデアの獲得

### チャレンジイベント
- ⚡ 厳しい納期: タイトなスケジュールでの完遂
- 🔥 困難な交渉: 難しいクライアント交渉
- 💪 チーム危機管理: プロジェクト危機の乗り越え
