# コンサルタント体験 人生ゲーム (MakeAnImpactThatMatters)

このリポジトリ向けに作った、30マスを経由して中心ゴールへ向かう「コンサルタント体験」人生ゲームのシンプルなブラウザ実装です。

特徴
- 30マスのスパイラル状ボード（外周から中心へ）
- ゴールはボードのど真ん中
- イベントマスは盤上に内容を書かず、プレイヤーが止まった時にそのプレイヤーの状態に合わせたイベントをフローティング（モーダル）で表示
- プレイ人数 2〜6人に対応
- ターン制（サイコロ：1〜6）

ファイル
- index.html — ゲーム UI
- styles.css — スタイル
- script.js — ロジック（プレイヤー、移動、イベント生成、UI制御）

使い方
1. index.html をブラウザで開きます（ローカルファイルで動作します）。
2. プレイヤー数を 2〜6 に設定し、プレイヤー名を入力して開始。
3. 「サイコロを振る」で順番に移動。イベントはプレイヤーの状態に応じて表示されます。

拡張案
- イベント内容のDB化や画像追加
- ネットワーク対戦／保存機能
- 応答パターンを増やして選択肢を与える（分岐）

実装上の注意
- イベントマス自体は盤上で空（マークのみ）。止まったときにのみプレイヤー状態に基づくイベントモーダルを表示します。
- 現在はクライアントサイドのみの実装です。
